# ระบบยืม-คืนอุปกรณ์กีฬา หน้าต่างสำหรับนักศึกษา (Sports Lending System - Student)
โปรเจกต์ระบบบริการยืมอุปกรณ์กีฬาออนไลน์ผ่านเว็บแอปพลิเคชันแบบ Realtime สำหรับนักศึกษามหาวิทยาลัยกรุงเทพ เชื่อมต่อโดยตรงกับระบบคลังข้อมูลส่วนกลาง

* **Link GitHub Repository:** https://github.com/irawansrasuphan/bu-sports
* **Link สำหรับเข้าใช้งานระบบ (GitHub Pages):** https://irawansrasuphan.github.io/bu-sports/

---

## ฟังก์ชันการทำงานหลัก (Core Features)
1. **Realtime Stock Browsing:** นักศึกษาสามารถเช็คจำนวนอุปกรณ์กีฬาที่ "พร้อมใช้งานจริง" ในคลังได้ทันที ก่อนจะเดินไปทำเรื่องยืมที่ห้อง
2. **Interactive Equipment Selection:** ระบบเลือกการ์ดอุปกรณ์กีฬาที่ต้องการยืมในรูปแบบมินิมอล (Minimalist Grid Cards) แสดงผลเข้าใจง่าย รองรับอุปกรณ์ 6 ประเภทหลักตามหน้างานจริงได้แก่:
   * ลูกฟุตบอล
   * วอลเลย์บอล
   * บาสเกตบอล
   * ปิงปอง (แพ็ค)
   * แบดมินตัน (ไม้)
   * เทนนิส
3. **Digital Borrow Request:** ฟอร์มส่งคำขอยืมแบบดิจิทัลเพื่อลดการใช้กระดาษ โดยระบุข้อมูลสำคัญครบถ้วน:
   * รหัสนักศึกษา (สำหรับใช้เป็นข้อมูลหลักในการอ้างอิงรหัสเฉพาะบุคคลที่ไม่มีวันซ้ำกัน)
   * คณะ / ภาควิชา (ในรูปแบบตัวเลือก Dropdown ป้องกันการพิมพ์ข้อมูลผิดพลาด)
   * กำหนดส่งคืน (ระบุเวลาที่คาดว่าจะนำอุปกรณ์มาคืนเจ้าหน้าที่)
4. **Instant Central Updates:** ทันทีที่นักศึกษากดยืนยันการยืม ระบบหลังบ้านจะหักยอดสต็อกในคลัง และส่งรายชื่อผู้ยืมพร้อมเวลาคืนไปปรากฏบนแดชบอร์ดของผู้ดูแลระบบ (Admin) ทันทีโดยไม่ต้องโหลดหน้าเว็บใหม่

## 💻 เทคโนโลยีที่เลือกใช้ (System Prerequisites & Libraries)
ตัวแอปพลิเคชันถูกออกแบบมาให้เป็นมิตรกับสมาร์ทโฟนและคอมพิวเตอร์ (Responsive Design) เพื่อให้นักศึกษาสามารถใช้งานผ่านเบราว์เซอร์ได้สะดวกสบาย
* **Frontend:** HTML5, CSS3 (เน้นโทนสีครีม-เหลืองพาสเทล มินิมอล อบอุ่นและใช้งานง่าย) และ Vanilla JavaScript
* **Database:** Firebase Realtime Database สำหรับการประมวลผลจัดเก็บข้อมูลและส่งสัญญาณข้ามฝั่งระหว่างนักศึกษากับแอดมิน

## 🛠️ ขั้นตอนการนำไปใช้และทดสอบโปรแกรม (How to Run & Test)

### วิธีที่ 1: ทดสอบผ่านระบบออนไลน์ (สำหรับนักศึกษาและผู้ตรวจ)
1. คลิกเข้าสู่ลิงก์บริการผ่าน GitHub Pages ฝั่งนักศึกษา: `https://irawansrasuphan.github.io/bu-sports/`
2. เลือกรหัสอุปกรณ์กีฬาที่ต้องการยืม (การ์ดจะเปลี่ยนสถานะเพื่อแสดงว่าถูกเลือกแล้ว)
3. กำหนดเวลาที่จะส่งคืนให้เรียบร้อย
4. กดปุ่ม **"ยืนยันการส่งคำขอยืม"**
5. *ผลลัพธ์ที่เกิดขึ้น:* ตัวเลขจำนวนอุปกรณ์คงเหลือบนหน้าจอจะลดลงอัตโนมัติ และเมื่อเปิดหน้าแดชบอร์ดฝั่งแอดมิน รายชื่อและเวลากำหนดส่งคืนจะไปแสดงผลในกล่องผู้ค้างส่งทันที

### วิธีที่ 2: รันโปรแกรมจาก Source Code บนเครื่องคอมพิวเตอร์ (Local Setup)
1. ดาวน์โหลด Source Code โปรเจกต์ฝั่งนักศึกษาลงเครื่องคอมพิวเตอร์และทำการแตกไฟล์ ZIP
2. ตรวจสอบไฟล์โครงสร้างหลักระบบ ซึ่งภายในโฟลเดอร์หลักจะต้องประกอบไปด้วยไฟล์ 3 ชิ้นดังนี้:
   ├── index.html   # หน้าโครงสร้างฟอร์มและการ์ดเลือกอุปกรณ์สำหรับนักศึกษา
   ├── style.css    # ไฟล์คุมธีมสีเหลืองครีมและหน้าต่างการ์ดไอเทม
   └── script.js    # ไฟล์ประมวลผลคำสั่งยืมและยิงข้อมูลไปหา Firebase API
3. สามารถเปิดทดสอบบนเครื่องได้ทันทีด้วยการดับเบิ้ลคลิกไฟล์ `index.html` ผ่านเว็บเบราว์เซอร์ทั่วไป

## 🗄️ การจัดเก็บโครงสร้างข้อมูลเมื่อเกิดการยืม (Data Transaction Example)
